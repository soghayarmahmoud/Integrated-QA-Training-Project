# Support

If you need assistance with the Integrated QA Training Project, the best place to start is by opening an issue.

## Ask for help

- Use the issue tracker for questions about project setup, test execution, and documentation.
- Provide details about your environment and what you have tried.
- Reference the relevant documentation section when possible.

## Getting assistance

- `README.md` for general project guidance
- `CONTRIBUTING.md` for contribution workflow
- `SECURITY.md` for vulnerability reporting

## Expected response

Maintainers will review support requests as soon as possible. If you need urgent help, include a clear subject line and a concise description.
